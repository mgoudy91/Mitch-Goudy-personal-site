Senior Project: MMS Visualizer

For my Senior Project at CU, I was able to create a visualizer for NASA and LASP’s joint MMS (Magnetospheric Multiscale) mission along with my classmates Nick Aberle, Taylor Hall, Theodore Hoff, and Dane Larsen.

The visualizer is essentially a web-application made to be accessible to operations and science team members of the MMS project, as well as any space enthusiasts in the general public interested in the project.

The client, visualizer, and user interface is written primarily in JavaScript, notably webGL and jQuery, and is supported by a web.py python back-end in change of information grabbing, massaging, and serving to the client.

My role in the project was Team Lead, as well as designer of most of the user interface and some of the JavaScript client-side logic.

Check it out!

JamWalkr

JamWalkr was originally a project I made with Ivona Andrzejewski, Casey Chevalier, and Patrick Vargas for our Software Engineering Methods and Tools final project. In short, it’s a web-based music streaming application, with an interesting geographic component. Users vote on the kinds of music that are popular in certain areas, and the most popular type of music in each area is used to create playlists and then streamed on demand.

It is written in PHP, JavaScript, jQuery, HTML, CSS, and mySQL. It also uses the Twitter Bootstrap and Google Maps API.

Recently, I’ve been working on making the application more mobile friendly, and even won third place in CU’s Mobile App Challenge!
